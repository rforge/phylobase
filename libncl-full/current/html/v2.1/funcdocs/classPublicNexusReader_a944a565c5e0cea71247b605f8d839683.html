NxsUnalignedBlock * PublicNexusReader::GetUnalignedBlock ( const NxsTaxaBlock *  taxa,
unsigned  index 
) const
Returns:
a pointer to the NxsUnalignedBlock with index Indexing starts at 0 and refers to the index in a list of NxsUnalignedBlock objects that refer to the NxsTaxaBlock `taxa`. Thus, the index does not necessarily represent the position among ALL of the NxsUnalignedBlock objects

0L will be returned if the index is out of range. Indices should be < the number returned by GetNumUnalignedBlocks(taxa). If `taxa` is 0L, then the total block indexing scheme will refer to the total number of Unaligned blocks read.
