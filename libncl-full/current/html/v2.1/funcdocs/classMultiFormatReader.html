Detailed Description

A special class of PublicNexusReader, that can parse.

  • PHYLIP,
  • relaxed PHYLIP,
  • FASTA, and
  • ALN formatted files in addition to NEXUS. Non-NEXUS files are parsed and the information from these files is added to the appropriate NxsBlock object. So the parser essentially creates a the normal NCL interface even if the input is not NEXUS
