bool NxsSetReader::AddRange ( unsigned  first,
unsigned  last,
unsigned  modulus = 0 
) [protected]

Adds the range specified by `first', `last', and `modulus' to the set.

If `modulus' is zero it is ignored. The parameters `first' and `last' refer to numbers found in the data file itself, and thus have range [1..`max']. They are stored in `nxsset', however, with offset 0. For example, if the data file says "4-10\2" this function would be called with `first' = 4, `last' = 10 and `modulus' = 2, and the values stored in `nxsset' would be 3, 5, 7, 9. The return value is true unless `last' is greater than `max', `first' is less than 1, or `first' is greater than `last': in any of these cases, the return value is false to indicate failure to store this range.
